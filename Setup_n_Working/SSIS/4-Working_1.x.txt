--create a table that will allow us to watch for changes in our control loop variable

CREATE TABLE [dbo].[tblForLoopExample](
	[LoopExampleKey] [int] PRIMARY KEY IDENTITY(1,1) NOT NULL,
	ControlLoopVariableValue int,
	[LastUpdated] [datetime] 
)

In SSIS, we need to create a variable for our counter.
We will add a variable named Counter with the Int32 data type.
we will use the For Loop Container to repeat an Execute SQL Task

execute sql task > insert into dbo.tblForLoopExample values (@Counter, getdate())

Click on Parameter Mapping on the left side of the Execute SQL Task Editor to display the Parameter Mapping page of the 
Execute SQL Task Editor. In the Variable Name column select User::Counter (it should be at or near the bottom of the drop-down list).
 Select Input as the Direction, Int32 as the Data Type, and enter "Counter" for the Parameter Name. We will leave
 the Parameter Size at -1. 

Now add 'For loop' and add execute sql task to it.
